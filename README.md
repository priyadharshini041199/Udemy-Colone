# Udemy Clone (HTML & CSS)

This project is a Udemy clone website created using HTML and CSS, designed to replicate the visual layout and user interface of the original Udemy platform. It focuses on building a clean, responsive, and user-friendly front-end using modern CSS techniques.

The website includes a structured homepage with a navigation bar, course categories, featured courses displayed as cards, and a footer section. This project was developed to strengthen core front-end skills such as layout design, responsive styling, and semantic HTML.

Features

Responsive design for different screen sizes

Navigation bar with branding and menu links

Course cards with images, titles, ratings, and pricing

Category and featured course sections

Modern UI using pure HTML and CSS

Technologies Used

HTML5

CSS3 (Flexbox & Grid)

Purpose

This project is created for educational and practice purposes only. It does not include backend functionality or real data and is not affiliated with Udemy.
